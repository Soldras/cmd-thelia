# Global script

## T_cc

Allow you to clear cache from all env

### Usage

T_cc.sh [-h] [-e ENV]

### Options
	-h display this message
	-e set the environment to clear

## T_right

Quick change right for Thelia in dev

### Usage

T_right.sh [-h] [-c] CODE

### Options
	-h display this message
    -c set code for chmod ( default : 777 )

## T_psr

Fix psr-2 on Thelia modules

### Usage

T_psr.sh [-h] -m MODULE

### Options
     -h display this message
     -m set module name
